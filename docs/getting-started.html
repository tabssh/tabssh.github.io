Getting Started with TabSSH

Download, install, and connect to your first server in minutes.

Installation

GitHub Releases (Primary Method)

  1. Visit our download page for the latest release
  2. Choose your device architecture (ARM64 for most devices)
  3. Download the APK (filename: tabssh-android-{arch}.apk)
  4. Enable "Install from unknown sources" in Android settings
  5. Install the APK and launch TabSSH

First Connection

Quick Connect

  1. Tap the "+" button in TabSSH
  2. Enter your server details (host, port, username)
  3. Choose authentication (password or SSH key)
  4. Tap "Connect"
